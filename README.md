# Etch-a-Sketch

This is a simple Etch-a-Sketch drawing pad built with HTML, CSS (Flexbox), and vanilla JavaScript.

The project is part of The Odin Project Foundations curriculum and focuses on DOM manipulation, event handling, and dynamic layout creation.

## Features

- 16x16 grid generated dynamically with JavaScript
- Draw by hovering the mouse over the grid squares
- Button to reset the grid with a custom size (up to 100x100)
- Fixed drawing area size (960px x 960px)

## How to use

1. Open `index.html` in your browser
2. Move the mouse over the grid to draw
3. Click the button and choose a new grid size to reset the drawing area
